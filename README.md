# course-plus

SJTU 学期开课表索引。

## 使用方法
### 在线使用
本项目已部署至 GitHub Pages ，网址： https://sjtu-plus.github.io/course-plus/


### 从代码运行

软件需求

[Node.js](https://nodejs.org/)

克隆存储库
```
git clone https://github.com/SJTU-Plus/course-plus.git
cd course-plus
git submodule init
git submodule update
```

安装依赖包
```
npm i
```

启动本地服务
```
npm run serve
```
